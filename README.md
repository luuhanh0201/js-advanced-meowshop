## GITHUB
```
git add <file>: Thêm một tập tin hoặc thay đổi vào chỉ mục (staging area) để chuẩn bị cho việc commit.

git status: Hiển thị trạng thái của kho chứa và các tập tin trong thư mục làm việc.

git commit -m "<message>": Tạo một commit mới với thông điệp (message) mô tả các thay đổi đã thực hiện.

git push: Đẩy các commit đã tạo lên kho chứa trên GitHub.

git pull: Cập nhật các thay đổi mới nhất từ kho chứa trên GitHub về máy tính.

git checkout <branch>: Chuyển đổi sang một nhánh khác trong kho chứa.

git merge <branch>: Kết hợp các thay đổi từ một nhánh vào nhánh hiện tại.

git log: Hiển thị lịch sử commit.

```
#EX:
```
    - Hợp nhất code từ nhánh phụ vào dev:   - git checkout dev
                                            - git pull origin dev
                                            - Kiểm tra và giải quyết xung đột nếu có
                                            - git push origin dev

```
